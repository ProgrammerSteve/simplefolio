Projects

Smartbrain App

Started off as a udemy project, it is a react app that utilizes the clarifai API to detect faces in photos. Simply input the image url to activate. A ranking and badge(utilizes AWS lambda) is displayed to signify the number of uses. The information is kept in a postgresql database on AWS. The frontend is react while the backend is nodejs. The frontend code is deployed on AWS S3 while the backend is in an EC2 instance.

Mandarin HSK1 Training

A react app that uses a psuedorandom number generator to continously display flash cards for HSK level 1 vocabulary. Inputs are handled by a reducer made with react hooks. There are 7 variables in the state for the main functional component. Input choice positions are randomized in the form element.

React Square Game

This app has 6 levels where your goal is to get the red circle to reach the yellow square while avoiding the purple square. React motion library is used to move the div element, useContext shares state between the game and score cards, useEffect/useRef is used to keep track of the time per stage, and the useEffect hook keeps track of any changes to the state in order to progress the game.

An app of functional and class components made throughout the udemy course that utilizes jest for unit testing react components with enzyme. The application also makes use of redux and redux-thunk for state management and is a progressive web app. Making use of an api call, the profile card information is uploaded and can be searched through using the search bar at the top.
